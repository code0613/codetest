# 코딩테스트 No.13
## 설명 
- 2016년 1월 1일은 금요일입니다. 2016년 a월 b일은 무슨 요일일까요? 두 수 a ,b를 입력받아 2016년 a월 b일이 무슨 요일인지 리턴하는 함수, solution을 완성하세요. 
- 요일의 이름은 일요일부터 토요일까지 각각 SUN,MON,TUE,WED,THU,FRI,SAT입니다.
- 예를 들어 a=5, b=24라면 5월 24일은 화요일이므로 문자열 "TUE"를 반환하세요.

## 제한 조건
- 2016년은 윤년입니다.
- 2016년 a월 b일은 실제로 있는 날입니다. (13월 26일이나 2월 45일같은 날짜는 주어지지 않습니다)

``` java
class Solution {
    public String solution(int a, int b) {
       String answer = "";
        String week[] = {
                "FRI","SAT","SUN","MON","TUE","WED","THU"
        };
        int payday = 0;

        int[] month = {31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

        for (int i = 0; i < a-1; i++){
            payday += month[i]; //a == 1 실행하지않고 넘어감
        }
        payday += b-1;
        payday = payday%7;
        answer = week[payday];
        return answer;
    }
}
```
## 문제점
처음 문제를 봤을때 어디서 부터 시작해야 될지 감이 잡히지 않았다<br>
고민후에 코드를 쓰기는 했지만 내 생각처럼 움직이는 코드가 되지않아서<br>
처음으로 도움을 요청했던 문제이다

- 16년의 1월1일은 금요일부터 시작이니 week[] 의 요일순서를 금-목으로 설정한다
- 변수 payday는 지난 날을 표시하며
<br>int[] month는 지나갈 날 즉 각 달의 일수를 요소로 가지고 있다
- for문을 돌며 payday에 month[i]를 추가한다 <br>a가 1일경우는 for문의 코드를 실행하지 않는다
<br> 찾는 요일이 1월달 일경우는 지나간 날을 추가하지 않는다
- payday에 payday를 7로 나눈 나머지를 넣으면 최초에 설정한 1월1일 금요일에서 며칠이 지난 요일인지 알수있게 된다
